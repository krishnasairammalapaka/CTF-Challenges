Basic SQL Injection Challenge

A small web application has a login page that is vulnerable to SQL injection.

The flag is stored in the database under the admin account. Your objective is to bypass the authentication by exploiting this vulnerability.

For example, try entering the following payload in the username field:

admin' OR '1'='1

Leave the password field as is (or provide any value) to successfully log in and retrieve the flag from the admin account's details.

Hint: Understand how SQL injection can bypass authentication. A simple approach is to inject a condition that always evaluates to true.

Good luck, and happy CTF-ing!
